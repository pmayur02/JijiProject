# üìö Learn with Jiji ‚Äì Backend Service

Backend service that powers **Jiji‚Äôs search & respond flow** for AI learning content.

This service:
1. Accepts a user query  
2. Retrieves relevant learning content  
3. Returns a structured response consumable by the frontend  


---

## üèóÔ∏è Tech Stack

- Node.js  
- Express.js  
- Supabase (DB + Auth + Storage)  
- dotenv  
- Swagger UI  

API Docs:  
http://localhost:8000/api-docs/#

---

## üì° API Endpoints

POST /register   ‚Üí Register user  
POST /login      ‚Üí Login user
